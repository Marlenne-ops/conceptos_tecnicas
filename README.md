# conceptos_tecnicas
Es el lugar donde Git almacena los archivos y su historial de cambios.  Repositorio local: en tu computadora.  Repositorio remoto: en un servidor como GitHub.
| **Comando**               | **Función**                                                        |
| ------------------------- | ------------------------------------------------------------------ |
| `git clone URL`           | Clona un repositorio remoto a tu computadora.                      |
| `cd nombre_carpeta`       | Entra a una carpeta específica.                                    |
| `ls`                      | Muestra los archivos y carpetas del directorio actual.             |
| `mkdir nombre_carpeta`    | Crea una nueva carpeta.                                            |
| `touch nombre_archivo`    | Crea un archivo vacío.                                             |
| `cd ..`                   | Regresa a la carpeta anterior.                                     |
| `git add .`               | Agrega todos los cambios al área de preparación (staging).         |
| `git commit -m "mensaje"` | Guarda los cambios en el historial del repositorio con un mensaje. |
| `git push origin main`    | Envía los commits al repositorio remoto en GitHub.                 |
|   git pull --rebase       | Trae los cambios del repositorio remoto (GitHub).                  |
